A downloadable game for Windows

※1 WINDOWS 11、並びスマホアプリ「JoiPlay」動作確認済 注：JoIPlayではシャットダウン時エラーバグ有

本格的にツクラーとしてデビューする、ずっと昔・・・。

約 20 年近く前になりますでしょうか。


RPGツクールVXで、とりあえずツクってみた RPG を

当時ノータッチだったRGSS関連を導入し、さらに色々入れて遊びやすくした上で

今回、出させていただくことになりました。


物語としては勇者ラルフ(名前固定)と

後々仲間になる仲間たちが、魔王城へ乗り込む

シリアス要素ほぼ無しな、デフォ戦 RPG となります。


色々駆使すれば、エンディングまで３時間近くで着けるかも！？


恐らくツクールVX（注：念のためVX Aceでは無い）としては最初にして、最後になるかもしれない作品となります。

ひとつ、よろしくお願いいたします。



余談ですが、この作品をツクってる頃はVIPRPGの存在を全く知りませんでした(汗)。

時期的にはプレシャスか玉露リレーの時期だったかな？

Install instructions

上記ゲームフォルダ「umn8vx_ralfvxrpg.zip」をダウンロード後、解凍ソフトを使用して下さい。

また、プレイ時に前述のとおり『RPGツクールVX ランタイムパッケージ(RTP)』が必須なので

ツクール公式からダウンロードし、解凍しセットアップしてください。

あとはフォルダ『umn8vx_ralfvxrpg』内にある、アイコン『Game.exe』を

ダブルクリックすると、ゲームが開始されます。

----------------------------------------------------------------------------------------------------

※スマホアプリ『JpiPlay』でプレイする場合 ↓

スマートフォンでＲＰＧツクールＭＶ及びＭＺをプレイ出来るエミュレーター

『MaldiVes』を使用する際は、以下の手順で『MaldiVes』に登録の上プレイしてください。

『Add new game』をタッチ

↓

英語の注意書きが表示されＯＫをタッチ

↓

スマホ内に入れた当フォルダ『proj_artur_light』内の『www』フォルダをタッチ、

『www』フォルダ『このフォルダを使用』をタッチする

↓

この作品のアイコンとタイトル名が画面に追加されたのを確認したら

この作品のアイコンをタッチ、『ＲＵＮ ＧＡＭＥ』をタッチ。

以上でプレイが出来るはずです。
